# Home-Lab
First PC build - home server for virtual machines (VMs), network-attached server (NAS)

Overview

This home lab is designed for learning, experimentation, and practical applications, including virtualization, storage, networking, and potential future IoT/home security projects. The primary goal is to keep costs low while ensuring flexibility for future upgrades.

Planned Use Cases

Running virtual machines (VMs) accessible over the internet

Storing and remotely accessing large RAW photo files

Hosting a small Plex server (low priority)

Setting up an access point after moving

Installing a firewall (on a separate device)

Future potential IoT home security automation
